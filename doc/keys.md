### g

- `gb` - open github page from user/repo
- `gc` - commenting
- `gl` - Go Look/LSP
- `gs` - substutite
- `gy` - yank and comment line

### v

- vm vo vq `vv` vz

### z/Z

- zq is free
- **ZQ** = `:q!`
- **ZZ** = `:x` = `:wq`, but write only when changes have been made
